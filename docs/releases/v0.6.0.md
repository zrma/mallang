# Mallang v0.6.0 Release Notes

## Status

Mallang 0.6.0 is published as a GitHub source release:

```text
https://github.com/zrma/mallang/releases/tag/v0.6.0
```

Crates.io publication remains disabled.

## Highlights

- Compiler-owned `std/strings`, `std/errors`, `std/os`, `std/io`, `std/fs`,
  and `std/collections` packages provide the first standard-library surface.
- UTF-8 text, process arguments and environment, standard streams, and file I/O
  use owned values and recoverable `Result` errors.
- Opaque `Map[K,V]` supports deterministic hashing, replacement, removal, and
  call-scoped read and update callbacks.
- The multi-module `textstats` reference CLI exercises the complete native
  project and standard-library path.
- Canonical acceptance covers macOS arm64 and Ubuntu Linux x86_64.
