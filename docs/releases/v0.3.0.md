# Mallang v0.3.0 Release Notes

## Status

Mallang 0.3.0 is published as a GitHub source release:

```text
https://github.com/zrma/mallang/releases/tag/v0.3.0
```

Crates.io publication remains disabled.

## Highlights

- Function type syntax, named function values, and indirect calls make
  functions first-class values.
- Owned closure captures preserve move-only value safety.
- Mutable and nested closures retain explicit access and cleanup rules.
- Package-qualified functions can cross package APIs as values.
