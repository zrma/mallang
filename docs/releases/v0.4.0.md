# Mallang v0.4.0 Release Notes

## Status

Mallang 0.4.0 is published as a GitHub source release:

```text
https://github.com/zrma/mallang/releases/tag/v0.4.0
```

Crates.io publication remains disabled.

## Highlights

- Generic structs, functions, receiver methods, and package APIs use explicit
  type arguments.
- User-defined generic enums support exhaustive nested patterns.
- Deterministic specialization feeds typed IR and native C lowering.
- Generic declaration and specialization failures retain source diagnostics.

and release-harness-only commit.
