# Mallang v0.7.0 Release Notes

## Status

Mallang 0.7.0 is published as a GitHub binary release:

```text
https://github.com/zrma/mallang/releases/tag/v0.7.0
```

Crates.io publication remains disabled.

## Highlights

- `mlg fmt` provides deterministic comment-preserving formatting and no-write
  `fmt --check` verification.
- `mlg test` discovers project tests, preflights the complete suite, and runs
  each selected test in an isolated native process.
- Manifest-relative local path dependencies support deterministic project
  graphs and entrypoint-free library check/test workflows.
- `mallang.diagnostic.v1` JSON Lines diagnostics preserve human-output parity,
  project-relative paths, UTF-8 spans, and stable compiler stages.
- Deterministic release archives and `install.sh` support macOS arm64 and Linux
  x86_64 with SHA-256, archive-shape, and staged-version verification.
- Clean-project acceptance exercises format, check, test, build, install, and
  run using only the installed release compiler.

## Install

```sh
curl -fsSLO https://github.com/zrma/mallang/releases/download/v0.7.0/install.sh
chmod +x install.sh
./install.sh --version 0.7.0
```

`clang` is required for Mallang native build, run, and test commands.
