# Mallang v1.0.0-rc.1 Release Notes

## Status

Mallang 1.0.0-rc.1 is published as a GitHub binary prerelease:

```text
https://github.com/zrma/mallang/releases/tag/v1.0.0-rc.1
```

It is not the stable v1.0.0 release. Crates.io publication remains disabled.

## Highlights

- The RC implements the unchanged 98-rule v1 candidate frozen in v0.9.0.
- Release archives, checksums, installer verification, and `mlg --version` now support the exact
  SemVer prerelease version `1.0.0-rc.1`.
- The canonical rehearsal installs published v0.9.0, upgrades the same prefix to the RC, rolls
  back explicitly to v0.9.0, and re-upgrades to the RC.
- Clean-copy `textstats` format, check, test, build, run, output, error flow, strict C, sanitizer,
  and allocation accounting remain identical across baseline, upgrade, rollback, and re-upgrade.
- macOS arm64 and Linux x86_64 run the complete freeze, release compiler, generated C sanitizer,
  clean install, upgrade, and rollback acceptance before producing the combined checksum bundle.

## Install

```sh
curl -fsSLO https://github.com/zrma/mallang/releases/download/v1.0.0-rc.1/install.sh
chmod +x install.sh
./install.sh --version 1.0.0-rc.1
```

Rollback explicitly with the same installer:

```sh
./install.sh --version 0.9.0
```

`clang` is required for Mallang native build, run, and test commands.

## Verification

```sh
scripts/check-v1-rc-acceptance.sh
scripts/verify-v0-rc.sh
```

The signed prerelease tag points at the synchronized RC specification, P172 rehearsal record,
package version, release notes, and supported-platform artifact contract.
