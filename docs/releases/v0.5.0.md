# Mallang v0.5.0 Release Notes

## Status

Mallang 0.5.0 is published as a GitHub source release:

```text
https://github.com/zrma/mallang/releases/tag/v0.5.0
```

Crates.io publication remains disabled.

## Highlights

- Multi-payload and recursive enums use compiler-owned storage with recursive
  cleanup.
- Full-expression temporaries and direct `con`/`mut` call boundaries close
  ownership gaps without first-class references.
- Static and owned strings share immutable value semantics.
- Allocation accounting and deterministic failure injection cover every
  compiler-owned heap allocation family.
