# Mallang v1.0.0 Release Notes

## Status

Mallang 1.0.0 is the first stable GitHub binary release:

```text
https://github.com/zrma/mallang/releases/tag/v1.0.0
```

Crates.io publication remains disabled.

## Highlights

- The stable release implements the unchanged 98-rule language contract frozen
  in v0.9.0 and rehearsed by v1.0.0-rc.1.
- Compiler source and the conformance map are unchanged from v0.9.0.
- The 1.x source, observable-semantics, standard-library, CLI, diagnostic,
  installer and supported-platform compatibility guarantees now apply.
- The final rehearsal upgrades published v1.0.0-rc.1 to stable, rolls back to the
  RC, and re-upgrades while preserving representative `textstats` behavior.
- macOS arm64 and Linux x86_64 run the complete compiler, sanitizer, release and
  installation acceptance before producing the combined checksum bundle.

## Install

```sh
curl -fsSLO https://github.com/zrma/mallang/releases/download/v1.0.0/install.sh
chmod +x install.sh
./install.sh --version 1.0.0
```

`clang` is required for Mallang native build, run and test commands.

## Verification

```sh
scripts/check-v1-stable-acceptance.sh
scripts/verify-v0-rc.sh
```

The signed tag points at the synchronized stable specification, compatibility
policy, final acceptance record, package version and supported-platform artifact
contract.
