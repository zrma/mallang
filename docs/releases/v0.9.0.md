# Mallang v0.9.0 Release Notes

## Status

Mallang 0.9.0 is published as a GitHub binary release:

```text
https://github.com/zrma/mallang/releases/tag/v0.9.0
```

Crates.io publication remains disabled.

## Highlights

- The implemented v1 candidate is indexed as 98 normative source, type, ownership, standard
  package, CLI, diagnostic, runtime, distribution, and compatibility rules.
- A fail-closed conformance map connects every rule exactly once to 23 evidence profiles and 64
  scripts, fixtures, Rust tests, or explicit verification commands.
- The v1 compatibility policy defines compiler/language versioning, 1.x source and observable
  semantics guarantees, release classes, deprecation, narrow soundness exceptions, and no edition.
- The migration guide and executable fixtures cover canonical `con`/`mut`, range ownership,
  removed bootstrap syntax, project workflow, standard packages, and tooling.
- The `textstats` representative project repeatedly passes format, check, test, build, run,
  strict C, sanitizer, allocation accounting, output, and error-flow gates from a clean install.
- The freeze audit confirms no compiler source changes after v0.8.0; all intervening changes are
  documentation, conformance, dogfood, or release work.

## Install

```sh
curl -fsSLO https://github.com/zrma/mallang/releases/download/v0.9.0/install.sh
chmod +x install.sh
./install.sh --version 0.9.0
```

`clang` is required for Mallang native build, run, and test commands.
