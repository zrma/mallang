# Mallang v0.2.0 Release Notes

## Status

Mallang 0.2.0 is published as a GitHub source release:

```text
https://github.com/zrma/mallang/releases/tag/v0.2.0
```

Crates.io publication remains disabled.

## Highlights

- Multi-file compilation units preserve file-aware diagnostics.
- `mallang.toml`, package declarations, imports, and project discovery define
  the project model.
- Import graph validation and cross-package semantic linking support public
  functions, structs, and receiver methods.
- `mlg check`, `mlg build`, and `mlg run` accept projects while preserving
  standalone `.mlg` operation.
