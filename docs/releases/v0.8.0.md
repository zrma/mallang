# Mallang v0.8.0 Release Notes

## Status

Mallang 0.8.0 is published as a GitHub binary release:

```text
https://github.com/zrma/mallang/releases/tag/v0.8.0
```

Crates.io publication remains disabled.

## Highlights

- Bounded parser recovery emits deterministic multiple diagnostics while rejecting partial
  programs before later compiler stages.
- User-reachable compiler invariants and malformed typed IR return owned diagnostics instead of
  relying on unchecked backend assumptions.
- Stable-toolchain lexer/parser/type/ownership properties and a minimized crash corpus run in
  canonical and optimized release-binary gates.
- Four representative programs have a machine-readable observational performance baseline with
  stable runtime output and generated C hashes.
- Generated C and deterministic release archives have explicit same-input byte-identity gates.
- The complete hardening acceptance runs on macOS arm64 and Linux x86_64 release artifacts.

## Install

```sh
curl -fsSLO https://github.com/zrma/mallang/releases/download/v0.8.0/install.sh
chmod +x install.sh
./install.sh --version 0.8.0
```

`clang` is required for Mallang native build, run, and test commands.
